# Project 5: E-commerce Analytics

## Objective
To analyze e-commerce data for customer segmentation, purchase pattern analysis,
and recommendation insights.

## Datasets Used
- customers.csv: Customer demographics
- products.csv: Product categories and prices
- orders.csv: Purchase history

## Methodology
- Data loading and preprocessing using Pandas
- Customer segmentation using K-Means clustering
- Category-wise and time-based purchase analysis
- Identification of top-selling products

## Tools & Technologies
Python, Pandas, Matplotlib, Scikit-learn, Jupyter Notebook

## Key Findings
- Customers fall into three major segments
- Some categories dominate overall sales
- Few products drive most purchases

## Conclusion
This project helps businesses improve marketing strategies,
inventory management, and personalized recommendations.
